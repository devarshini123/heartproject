 # HEART DISEASE PREDICTION

## Project Overview

Heart Disease Prediction is a Machine Learning classification project that predicts whether a patient is likely to have heart disease based on various medical attributes such as age, sex, chest pain type, cholesterol level, blood pressure, maximum heart rate, and other clinical parameters.

The objective of this project is to assist in the early detection of heart disease by analyzing patient health data and providing accurate predictions using machine learning models.


##  Problem Statement

Develop a machine learning model that predicts whether a patient has heart disease based on medical and clinical features.
